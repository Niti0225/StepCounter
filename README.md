# Stridevo Steps Website

Offizielle Website fuer die App **Stridevo Steps**.

Diese Website enthaelt:

- moderne Startseite
- App-Mockup
- Feature-Bereich
- Premium-/Pro-Vorschau
- Datenschutzseite
- Nutzungsbedingungen
- FAQ
- Impressum-Platzhalter

## Upload auf GitHub Pages

Lade den kompletten Inhalt dieses Ordners in dein GitHub-Repository hoch:

```text
github-upload-website
```

Wichtig: Die Datei `.nojekyll` sollte mit hochgeladen werden, damit GitHub Pages die Website ohne Jekyll verarbeitet.

## Dateien

```text
index.html
datenschutz.html
nutzungsbedingungen.html
faq.html
impressum.html
robots.txt
.nojekyll
UPLOAD-HINWEISE.txt
assets/
```

## Noch vor der echten Veroeffentlichung ersetzen

- Impressum mit echten Anbieterangaben
- echter App-Store-Link
- finale Abo-/RevenueCat-Informationen
- finale Werbe-/AdMob-Informationen
- App-Store-Datenschutzangaben passend zur finalen App-Version

## Datenschutz

Die Website verwendet keine externen Tracking-Skripte, keine CDN-Abhaengigkeiten und keine Analytics-Skripte.

Die Rechtstexte beschreiben die App aktuell als lokale Schrittzaehler-App mit:

- lokaler Speicherung
- keinem Konto
- keinem Cloud-Sync
- nicht-personalisierter Werbung
- Pro-/Abo-Vorschau bis zur echten Store-Anbindung
